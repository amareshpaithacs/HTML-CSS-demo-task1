# SnapX Photography Contest Website

A responsive photography contest landing page built using pure HTML and CSS.

## Features

* Responsive navigation bar
* Hero section with image cards
* Auto-changing image slider using CSS animations
* Contest categories section
* Winners section
* Pricing plans section
* Hover effects and responsive design

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
project/
│── index.html
│── style.css
│── images/
```

## How to Run

1. Download or clone the project
2. Open the project folder in VS Code
3. Run `index.html` using Live Server
